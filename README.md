# mine_detection
# 自動收集圖片與 TXT 檔案工具

這是一個純前端網頁工具，可以從指定的大資料夾中遞迴掃描所有子資料夾，並自動收集符合條件的圖片檔與 `.txt` 檔，最後輸出到使用者指定的新資料夾。

## 功能

- 選擇來源資料夾後，自動掃描所有子資料夾
- 支援圖片與標註檔案收集
- 支援副檔名：
  - `.jpg`
  - `.jpeg`
  - `.png`
  - `.bmp`
  - `.gif`
  - `.tif`
  - `.tiff`
  - `.webp`
  - `.txt`
- 顯示掃描結果：
  - 掃描資料夾數
  - 掃描總檔案數
  - 符合條件的檔案數
  - 掃描到的副檔名統計
- 按下 `Enter` 後，將符合條件的檔案複製到指定輸出資料夾
- 若遇到同名檔案，會自動重新命名，避免覆蓋

## 使用需求

此工具使用瀏覽器的 File System Access API，因此建議使用：

- Google Chrome
- Microsoft Edge

不建議使用 Firefox 或 Safari，因為可能不支援資料夾讀寫功能。

## 使用方式

1. 下載或複製本專案的 `index.html`
2. 使用 Chrome 或 Edge 開啟 `index.html`
3. 點選「選來源資料夾」
4. 選擇最外層的大資料夾
5. 等待掃描完成
6. 確認畫面顯示找到的檔案數
7. 按下鍵盤 `Enter`
8. 選擇輸出資料夾
9. 若瀏覽器要求權限，點選允許
10. 等待輸出完成
